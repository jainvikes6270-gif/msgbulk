LATHAEPS SMART v3.24.0 - ONE-TIME UPDATE SETUP

1. Keep LathaEPS_PRIVATE_SIGNING_KEY_BACKUP_v3_24_0.zip private and backed up.
2. Open its PRIVATE_GITHUB_SECRETS_SETUP.txt file.
3. In GitHub repository jainvikes6270-gif/msgbulk open:
   Settings > Secrets and variables > Actions > New repository secret
4. Add all four secrets exactly as shown in the private setup file.
5. Upload this public project to the main branch and wait for the green build check.
6. Open Releases > Draft a new release > create tag v3.24.0 from main > Publish.
7. The tag workflow publishes LathaEPS-Smart.apk and latest-version.json.

ONE-TIME MOVE FROM OLD DEBUG APK

1. In the old app use Settings > Local Backup and save the backup outside the app folder.
2. Uninstall the old debug-signed app.
3. Install the v3.24.0 LathaEPS-Smart.apk from the GitHub Release.
4. Restore the saved backup and re-enable Notification Access and Accessibility.
5. Activate the subscription again if the new permanent signing identity changes the Device ID.
