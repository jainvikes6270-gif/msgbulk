# Latha Bulk

Android app that loads phone contacts and opens WhatsApp chats one by one with a prefilled message. The user still taps **Send** inside WhatsApp. A PDF can also be selected and shared through WhatsApp.

## Build APK on GitHub
1. Upload the **contents of this folder** to the root of a GitHub repository.
2. Open **Actions**.
3. Open **Build Android APK**.
4. Tap **Run workflow**.
5. After the build finishes, open it and download the `LathaBulk-debug-apk` artifact.
6. Extract the artifact ZIP and install `app-debug.apk`.
